Insert a Image From Flickr in Octopress

昨天看了一下前天寫的文章，發現有一點難以閱讀，所以想說應該放些圖。說到放圖在 Blog 的方法的話，當然是先把圖放在圖床在 include 進來才是王道。剛好我有使用 Flickr，所以便開始 Survey 方法，結果找到了 這個網頁。

試了一下，發現這個 plugin 要插入 Original 的圖片時會有點問題 (VIP 錯了嗎？)，所以稍微研究了一下，發現 Original 的圖片透過 Flickr 的 API 的拿法跟其他大小的圖片會不太一下，所以自己把上面網頁提供的 API 更新如下：

require 'flickraw'                                                                                                                            
class FlickrImage < Liquid::Tag

  def initialize(tag_name, markup, tokens)
     super
     @markup = markup
     @id   = markup.split(' ')[0]
     @size = markup.split(' ')[1]
  end 

  def render(context)

    FlickRaw.api_key        = "140be6b9e097f6847b75f886379c5055"
    FlickRaw.shared_secret  = "b8fdaad2b4cacdcf"

    info = flickr.photos.getInfo(:photo_id => @id)

    server        = info['server']
    farm          = info['farm']
    id            = info['id']
    title         = info['title']
    description   = info['description']
    size          = "_#{@size}" if @size
    if @size == "o" then
        secret = info['originalsecret']
        format = info['originalformat']
    else
        secret = info['secret']
        format = "jpg"
    end 
    src           = "http://farm#{farm}.static.flickr.com/#{server}/#{id}_#{secret}#{size}.#{format}"
    page_url      = info['urls'][0]["_content"]

    img_tag       = "<img src='#{src}' title='#{title}'/>"
    link_tag      = "<a href='#{page_url}'>#{img_tag}</a>"

  end 
end

Liquid::Template.register_tag('flickrimage', FlickrImage)

經過這樣更改後要插入 Original 的圖片就沒有問題了。
